How to use

Auto OCR reads the page with no settings: line positions, baselines, font, size and spacing are measured from the pixels, and every line carries a byte-exactness certificate (green = certified, orange □ = honest unread). Redaction boxes and rules are detected and shown as overlays; the purple tick at each line's measured start should stack into the clean left edge real documents have — a quick visual read-quality check. Auto OCR All + Download .txt / .json export the whole document.
